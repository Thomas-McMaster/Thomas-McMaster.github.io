About

Hi. My name's Thomas. I'm an IT professional currently living in the West of Scotland and I specialize in data and database development.

I've been fortunate enough to hold data-related roles in the public, private, and charitable sectors, all working as a part of excellent teams. In each of those posts we've produced high quality work that has improved processes, gained value from organizational data, and kept decision-makers informed.

Most recently, I've been working in Azure on projects using Synapse Analytics, pySpark, and YAML-based pipelines. I'm Microsoft certified in Azure Fundamentals and I enjoy keeping up to date with data concepts and developments. I'm a keen road cyclist and speak A2 level Norwegian in line with the CEFR European language framework. I'm currently working towards B1 level certification, and plan to continue learning Norwegian as a working language.

Career experience:
  • Regularly working with data in efficient and performant ways
  • Data analysis that has assisted decision makers in important choices
  • Database administration based around security and reliable data restoration
  • Data migration experience at a variety of scales
  • Automation that's strongly improved existing business processes
Main technical skills:
  • SQL Server and T-SQL
  • Synapse Analytics, pySpark, and sparkSQL
  • CI CD through DevOps YAML-based pipelines
  • Powershell for automation of business tasks
  • Azure resources including Key Vault, RBAC, and DevOps
  • SQL Server Integration and Reporting Services

Data Analysis

Creating reports and dashboards for insight
