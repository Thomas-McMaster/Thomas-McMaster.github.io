About Me

My name's Thomas and I'm an IT professional living in the West of Scotland. I specialize in data and database development and, for over 15 years, I've worked with products including Microsoft SQL Server (2008 R2 to 2019), Integration Services, Reporting Services and PowerShell. I'm also Microsoft-Certified in Azure Fundamentals and have experience in Cloud offerings including Azure Data Factory and Synapse Analytics.

Since graduating from University I've been lucky enough to work in public, private and charitable sector roles. In each of those I've delivered:

  • strong SQL query knowledge that's extracted data in efficient and performant ways,
  • data analysis that's assisted decision makers in important choices,
  • database administration with a focus on security and reliable data restoration,
  • data migration at a variety of scales,
  • automation that's strongly improved business processes.

In my own time, I'm expanding my skills with Microsoft Power BI certification and learning Norwegian.
